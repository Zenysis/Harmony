Look at the [`<List>`](#list) component for more examples.

This is a basic list with hoverable items:

```jsx
import List from 'components/ui/List';

<List hoverableItems>
  <List.Item>
    First item
  </List.Item>
  <List.Item>
    Second item
  </List.Item>
  <List.Item>
    Third item
  </List.Item>
</List>
```
